## 기본 문법

| 입력 | 결과 |
|------|------|
| `# 제목1` | 큰 제목 |
| `## 제목2` | 중간 제목 |
| `### 제목3` | 작은 제목 |
| `**굵게**` | **굵게** |
| `*기울임*` | *기울임* |
| `` `코드` `` | `코드` |
| `> 인용` | 인용 블록 |
| `- 항목` | 목록 |
| `[링크](url)` | 클릭 가능한 링크 |
| `![이미지](url)` | 이미지 삽입 |
